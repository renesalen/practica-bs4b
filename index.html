deseo que el boton de jumbotron vaya a la derecha del párrafo, esto se tiene que hacer con flexbox

Debo agrupar el parrafo y el boton (selecionando ambos y crtl+shit+g)

A la clase de ese 'div' se le agrega d-flex ya que los elementos se forman en linea con flexbox

Para ponerlos a ambos extremos debemos utilizar justify-content-between

Se le encuentra en utilities → flex → justify-content.

Pero el parrafo y el boton no parecen estar alieneados, para ello lo alineamos con align-items-center

Deseo que el fondo del jumbotron sea fluido, pero contenido (texto) este del container para ello todo el contenido debe encerrarce dentro de una caja .container. esa caja '.container' debe de estar dentro de la caja div class'jumbotron'.

Si deseo cambiar de color de fondo le aplico bg-dark text-white, el mismo que se puede encontrar en: utilities → colors → background-color
